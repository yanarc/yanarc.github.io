Java 概述

发表于 2020-05-17 分类于 Java

Java 是什么

Java 是 Sun Microsystems 于1995 年⾸次发布的⼀种编程语⾔和计算平台。Java 主要分为三个版本
JavaSE(J2SE)(Java2 Platform Standard Edition， java平台标准版）
JavaEE(J2EE)(Java 2 Platform,Enterprise Edition， java平台企业版)
JavaME(J2ME)(Java 2 Platform Micro Edition， java平台微型版)。

Java 的特点

特点一：面向对象（Object Oriented）;
特点二：健壮性，摒弃了C++中难以理解的多继承、指针、内存管理等概念，不⽤⼿动管理对象的⽣命周期;
特点三：跨平台性，”Write once, run anywhere”;
特点四：功能强⼤和简单易⽤;
特点五：静态、编译性语言，执⾏效率要⽐动态语⾔⾼，速度更快;

Java 开发环境

理解JDK、JRE和JVM三者之间的关系
JDK：Java Development Kit Java 开发工具包
JDK = JRE + 开发工具集（javac.exe, java.exe, javadoc.exe）

JRE：Java Runtime Environpment Java 运行环境。包括Java虚拟机JVM和Java程序所需要的核心类库。
JRE = JVM + Java SE核心类库


开发环境主要是能够使得代码的编写，编译，运行能顺利执行。
